### Обзор

**Интро**

В проекте было необходимо сверстать адаптивный сайт о путешествии по России, используя макет
в Фигме. В работе использовался подход Desktop first.
При верстке сайта использовались следующие технологии:
* HtMl
* Css
* Flex
* Grid Layout
* Nested БЭМ
